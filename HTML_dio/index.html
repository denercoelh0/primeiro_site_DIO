O que é o HTML

HTML é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores. A tecnologia é fruto da junção entre os padrões HyTime e SGML. HyTime é um padrão para a representação estruturada de hipermídia e conteúdo baseado em tempo.

Aqui temos uma TAG h1, o que geralmente é usada como o titulo

Aqui uma tag P, o que seria um paragrafo.

A tag h2 serve como um subtitulo, ela é parecida com a h1, mas um pouco menor.

Aqui um paragrafo para mostrar algumas outras tag's temos aqui uma tag strong (serve para deixar tudo em negrito) e a tag u para sublinhar*. O asterisco está dentro da tag sup que deixa o que estiver dentro sobrescrito.

Aqui temos uma TAG i que deixa o texto em italico, dentro de uma tag blockquote, que deixa o texto com uma margem
